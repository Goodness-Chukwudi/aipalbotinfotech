Biometric DataBase System (Nigerian Prisons Service)

This system was designed as enterprise to manage all the states. Each state has local server which hosted MySQL Database. The central database at the headquarter is an SQL server to manage all entries from various states.

The MEPIN Central Database System

This application manages medical students courses and professional medical courses. The desktop app shared services with the web app (on the central server via VP) using WCF.

National Lawyers Verification Engine (MyLawsuit System)

The MyLawsuit System was developed for the National Lawyers to verify all the registered lawyers in the country. The client requested the design to be a windows application and still be able to protect the the National Lawyers detail from being copied or if copied not to be interpreted.

Integrated Enterprise Payroll System

This payroll system uses biometric registration to track duplicate registration/impersonation. It manages all features of the National Recommended Payroll System Policy, including TAX, NHIS, FGHB, EXEC, MEMBERS, Vouchers, Bank Schedule, Bank Reconciliation etc.

AipalStore Automated Inventory System

Client-Server Point of Sales software (also integrated with a web application) Clients => Local Server => Web app (Synchronized sales updates). This architecture was used to avoid system down-time when there is no internet connection, the POS can still function effectively and once there is connection the synchronization fires to update sales on the dedicated site through only sales update, mainly for the Boss or Manager to access daily sales reporting from the store. The POS also uses a Barcode scanner.
